Proyecto Cygni

Se parte del siguiente archivo cygni_edomex_backup_20220427_1223.sql el cual contiene la base de datos cygni_edomex con las siguientes tablas:

Las fuentes de los datos provienen de tres bases específicamente:

  • Listado de bases de datos SESNSP
  • Listado de bases de datos CONEVAL
  • Listado de bases de datos INEGI

A continuación se tiene el objetivo de usar Python para generar un dataframe general que reúna todas las tablas de la base de datos.

El script cygniEdomex.py realiza la conexión con Mysql Server para poder extraer la información de cygni_edomex, luego realiza el procesamiento de la información y organización en dataframes de acuerdo a la relación de las tablas. Finalmente se genera un dataframe general llamado Cygni. Este dataframe se convierte en un archivo binario para su utilización en el análisis posterior.
